PLAiCE - Python Desktop App Skeleton

This folder contains a minimal Tkinter-based desktop app skeleton.

How to run

- Run the app:

    python -m plaice_app

- Run the unit tests (uses stdlib unittest):

    python -m unittest discover -v

Notes

- The app uses Tkinter from the Python stdlib. On some platforms you may need to install system packages to provide the Tk GUI libraries.
